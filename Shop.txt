5️⃣ SPLIT PAYMENT (AUTOMATIC PAYOUTS)
✔ THIS IS THE KEY PART

WooCommerce + PayPal supports PayPal Commerce Platform split payments.

Each product must include:

Seller PayPal Merchant ID

Platform fee %

Payment Flow:

Buyer pays → PayPal splits →
✔ Artist paid instantly
✔ Platform keeps fee

{
  "purchase_units": [{
    "amount": { "value": "100.00" },
    "payee": { "merchant_id": "ARTIST_MERCHANT_ID" },
    "payment_instruction": {
      "disbursement_mode": "INSTANT",
      "platform_fees": [{
        "amount": { "value": "15.00" },
        "payee": { "merchant_id": "PLATFORM_MERCHANT_ID" }
      }]
    }
  }]
}
